## Como habilitar una API

1. Ir a [Google Cloud Console](https://console.cloud.google.com)
2. Seleccionar el proyecto
3. Menu → APIs y servicios → Biblioteca
4. Buscar la API → Habilitar
5. Crear credenciales (API Key u OAuth) segun corresponda
6. Actualizar esta tabla con ✅
